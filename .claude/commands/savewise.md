# /savewise 명령어

현재 대화에서 핵심 내용을 추출하여 Readwise에 하이라이트로 저장합니다.

## 사용자 요청

$ARGUMENTS

## 실행 규칙

1. **범위 확인**:
   - 이 대화에서 `readwise_save_highlights` 도구가 이전에 호출된 적이 있는지 확인
   - **있다면**: 마지막 호출 이후의 대화 내용만 분석
   - **없다면**: 전체 대화 내용 분석
2. **사용자 요청 반영**: 위 "사용자 요청"이 있다면 해당 주제/조건에 맞는 내용 위주로 추출
3. **대화 분석**: 해당 범위에서 저장할 가치가 있는 핵심 내용 3~7개를 추출
4. **추출 대상**:
   - 핵심 인사이트 및 개념
   - 결정사항 및 결론
   - 중요 명령어/코드 스니펫
   - 주의사항 및 경고
   - 유용한 팁 및 베스트 프랙티스
5. **readwise_save_highlights tool 호출**: 추출한 내용을 하이라이트 배열로 전달

## 하이라이트 형식

각 하이라이트는 다음 필드를 포함:
- `text`: 하이라이트 텍스트 (필수)
- `title`: "Claude 대화 - {주제}" 형식 (필수)
- `category`: "articles" (기본값)
- `note`: 컨텍스트 또는 추가 설명
- `tags`: 관련 태그 배열 (예: ["claude", "mcp", "typescript"])

## 실행

지금 대화를 분석하여 `readwise_save_highlights` tool을 호출하세요.
